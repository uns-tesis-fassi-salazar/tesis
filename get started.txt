
Pasos que realice al bajar el proyecto de nuevo (habiendo reinstalado windows):
    Antes de empezar tenia instalado lo siguiente:
        - Nodejs v12.14
        - npm v6.13.4
        - Angular cli 8.3.21
        - git
        - Arduino

- Bajar repo
    git clone https://github.com/uns-tesis-fassi-salazar/tesis

- Acciones para Angular
    $ cd tesis/angular-dashboard
    $ npm install
        NOTA: La primera vez me dio error por falta de permisos. Lo volvi a ejecutar con privilegios y no hubo problemas.
    $ npm start

Deploy de app web en Firebase por primera vez:
- Compilar app (genera archivos en /dist):
    $ ng build --prod
- Instalar las herramientas de consola de firebase:
    $ npm install -g firebase-tools
- Logearse e inicializar proyecto
    $ firebase login
    $ firebase init
- Completar preguntas
    > Database
- Realizar Deploy
    $ firebase deploy

Deploy de firebase:
    $ ng build --prod
    $ firebase deploy
